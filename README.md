## Tech Stack: Typescript, Graphql, Typeorm

## Install on local
- Set up Redis and Postgres
- Create a databse in Postgres
```
git clone https://github.com/w1216linw/blogserver.git
cd blogserver
yarn
```
create .env file, copy variables from .evn.example
```
yarn build
yarn dev
```
